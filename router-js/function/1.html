    # 瀑布流实现思路

    简单的实现方法：移动端常见的瀑布流分为两列（图文合并的卡片），等宽不等高，可把所有的列表数据拆分为左右两部分数据，先去获取 每一条数据的图片高度和宽度，图片设置等宽的像素，或者使用组件内置等宽比例（mode="widthFix"），得到宽高后，去计算每一张图片 的占比（height/width），如果左边比右边的占比小就把数据往左边 push，往下类推以长补短，先往高度短卡片的 push.

    布局：整一个列表大盒子居中，左右两边的列表小盒子给定宽比例，再给列表小盒子嵌套一层（在这里作循环渲染），所有图片设等宽不设高。左右两边代码一样，仅渲染的列表字段不同 lList rList

    //index.js
Page({
  data: {
    productListByTypeSort: [], //总列表
    lList: [],
    rList: [],
    lHeight: 0,
    rHeight: 0,
  },
  // 图片加载方法
  loadImaging({ currentTarget, detail }) {
    var lList = this.data.lList;
    var rList = this.data.rList;
    var lHeight = this.data.lHeight;
    var rHeight = this.data.rHeight;
    if (lHeight == rHeight || rHeight > lHeight) {
      lList.push(currentTarget.dataset.product);
      lHeight += detail.height / detail.width;
      this.setData({
        lList,
        lHeight,
      });
    } else {
      rList.push(currentTarget.dataset.product);
      rHeight += detail.height / detail.width;
      this.setData({
        rList,
        rHeight,
      });
    }
  },
});
